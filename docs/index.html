Abstract

This workshop explores molecular typography: designing type from small, reusable "typtom" elements. Starting from simple shapes on paper, we move towards digital prototypes and variable fonts, keeping the process accessible to designers and non-designers alike.

The focus is on systems, constraints, and experiments rather than polish: participants build a shared visual vocabulary and test it quickly across text, UI, and motion contexts.

Course plan

  • 01 Minimal theory — typtoms, modular grids, basic type parameters
  • 02 Choosing a concept — group brainstorming for the type family's character
  • 03 Paper prototyping — build glyph molecules with paper, stickers, or LEGO
  • 04 Physical to digital — translate modules into a font workspace
  • 05 Variable axes — define at least one axis (weight, width, mood)
  • 06 Testing in context — try the type in UI mockups, paragraphs, titles
