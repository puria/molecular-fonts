Workshop

Molecular Typography

A hands-on introduction to building type systems from simple modules, combining physical prototyping, variable fonts and minimal theory.

Abstract Course plan Timetable Materials & tools

Abstract

This workshop explores molecular typography: designing type from small, reusable “typtom” elements. Starting from simple shapes on paper, we will move towards digital prototypes and variable fonts, keeping the process accessible to both designers and non-designers.

The focus is on systems, constraints and experiments rather than polish: participants will build a shared visual vocabulary and test it quickly across text, UI and motion contexts.

Course plan / Synopsis

  • 01 — Minimal theory — What are typtoms, modular grids, and basic type parameters.
  • 02 — Choosing a concept — Group brainstorming for the “character” of our type family.
  • 03 — Paper prototyping — Build glyph molecules with paper, stickers or LEGO.
  • 04 — Physical to digital — Translate modules into a digital font workspace.
  • 05 — Variable axes — Define at least one simple axis (weight, width, mood...).
  • 06 — Testing in context — Try the type in UI mockups, paragraphs, and titles.
